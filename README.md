# AI Smart Water Management System

## Project Overview
This project is an AI-powered Smart Water Management System that monitors water usage, predicts future consumption using Machine Learning, and detects possible leakages. It provides an interactive dashboard developed using Streamlit.

## Features
- Water usage monitoring
- AI-based water usage prediction
- Leak detection
- Interactive dashboard
- Data visualization
- Smart insights

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Git & GitHub

## How to Run

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
